# OTP-Manager

> 运行在 Chrome 插件平台的 OTP 管理器。

## 使用方法

### 1. 打包
```shell
npm run bundle:prod
```

### 2. 打开 Chrome 的插件列表界面
[chrome://extensions/](chrome://extensions/)

开启右上角的 **开发者模式**

### 3. 加载插件
![usage](docs/assets/usage_1.png)

插件位于项目下的`dist`目录：
![usage](docs/assets/usage_2.png)
